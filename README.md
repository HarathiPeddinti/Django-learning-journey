# Django Learning Journey

This repository contains my Django practice projects and backend development exercises created while learning Django and REST API development.

## About

I created this repository to practice and improve my backend development skills using Django. It includes hands-on implementations of:

* Django project setup
* Django apps
* Models and migrations
* Views and URL routing
* CRUD operations
* REST API basics
* JSON responses
* Django ORM
* API testing
* Admin panel configuration

## How to Run the Project

```bash
git clone <repository_link>
cd project_folder

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
```
